# 地质生成与世界高度计划

## 当前 0.1.6 已处理

- 普通原版矿石 placed features 已通过 `neoforge:remove_features` 移除。
- 原版大矿脉不是 placed feature，而是 `OreVeinifier` 根据 `minecraft:ore_veininess` 噪声硬编码生成铜矿、铁矿和粗矿块；本版本覆盖该噪声为 0，避免自然出现原版铜矿和粗铜块。
- 矿床相对比例改为“煤和低品位铁较常见，普通铁较常见，铜中等偏少，金/青金石/红石/辰砂偏少，金伯利岩和含钻金伯利岩很稀少”。
- 新增围岩/伴生特征：磁铁矿带凝灰质围岩、黄铜矿带花岗质围岩、含金石英带方解石脉旁岩、金伯利岩带凝灰质角砾围岩、烟煤带砂岩沉积围岩、无烟煤带深板岩/变质围岩。
- 新增自然矿床来源：铝土矿风化壳、磷块岩沉积层、蒸发岩盐层、锡石砂矿、稀土碳酸岩矿体。它们先用稳定 placed feature 表达，再通过机器副产物模拟伴生矿。
- 普通石头、圆石、深板岩、深层圆石、黏土、砂、红砂、沙砾、泥土等都有处理路线；按住 Shift 可看到原版材料的近似组成。
- 旧测试存档若含有改名前的 `earth_online:*` 矿床方块，会通过 registry alias 映射到新的 `earth_on_minecraft:*`，避免进入旧区块时被替换成默认方块。

## 当前分布逻辑

| 类型 | 游戏分布 | 现实含义 |
| --- | --- | --- |
| 烟煤含煤岩 | 较常见，偏中高层 | 沉积盆地煤层，伴随砂岩类围岩，适合煤化工和早期燃料 |
| 无烟煤含煤岩 | 较少，偏中深层 | 变质程度更高的高碳煤，伴随深板岩类围岩，偏石墨/高热值路线 |
| 贫磁铁矿 | 常见，偏深部 | 低品位铁矿，保证开局可找到铁来源 |
| 普通/富磁铁矿 | 普通到稀少，偏深部 | 可用铁矿床主体，富矿明显更少 |
| 黄铜矿 | 中等偏少，跨浅深层 | 铜硫化矿，伴随花岗质/热液围岩和黄铁矿副产 |
| 含金石英、青金石、红石、辰砂 | 偏少 | 小型热液/变质/幻想矿物体，作为功能矿物入口 |
| 金伯利岩/含钻金伯利岩 | 稀少到极稀少 | 金刚石管状岩体，先用稀疏矿体表达，不做满地宝石 |
| 铝土矿风化壳 | 中等偏少，近地表 | 铝土矿/赤铁矿/高岭土共生，给铝工业更直接自然入口 |
| 磷块岩沉积层 | 偏少，中浅层 | 沉积磷矿，给磷肥、磷酸和石膏副产路线 |
| 蒸发岩盐层 | 偏少，中浅层 | 盐类沉积，给氯碱、盐卤、钾盐和硼酸盐路线 |
| 锡石砂矿 | 稀少，浅层 | 冲积/河床重矿物富集，给锡石、钛白粉和少量独居石 |
| 稀土碳酸岩 | 很稀少，偏中深层 | 稀土碳酸盐/磷酸盐矿体，给氟碳铈矿、独居石和少量铀矿物 |

这些比例不是直接照抄现实储量或年产量。现实比例要经过可玩性换算：基础燃料和低品位铁必须能支撑新手流程；铜要支撑机器但不能像煤一样泛滥；贵金属、宝石、辰砂、含钻金伯利岩需要稀少，否则失去探索意义。

## 仍需继续推进

- 真实矿产比例不能直接照搬全球储量或产量，否则 MC 生存会变成找不到资源；后续应按“现实相对丰度 + 玩家可玩性 + 兼容性需求”建立一张可调倍率表，并暴露到配置或世界预设。
- 伴生矿当前先用独立 placed feature、围岩 halo 和处理副产物表达。下一步可做专用矿床生成器，让主矿、伴生矿和围岩在同一个空间结构内成型，而不是多个独立随机点。
- 地层厚度确实比现实薄。默认 MC 26.2 主世界是 `min_y=-64,height=384`，直接硬改会影响旧世界和其他 mod；更安全的路线是提供新的世界预设或显式配置，再把矿床高度、岩性层、沉积层、深部变质带同步迁移。
- 后续可继续加入铅锌硫化矿床、沉积铁矿、石油/天然气盆地、盐丘形态、独居石海滨砂矿，以及更真实的“主矿 + 伴生矿 + 围岩 halo”同体矿床生成器。
