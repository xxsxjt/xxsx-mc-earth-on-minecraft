# Earth Online 贴图 AI 提示词

## 总体风格

所有贴图都应是原创像素风，适配 Minecraft 资源包，不直接仿照原版贴图。

通用正向提示词：

```text
original pixel art texture for a voxel sandbox game, 16x16 pixels, crisp pixel edges, hand-painted mineral detail, readable at small size, natural geological color variation, subtle dithering, no text, no watermark, no UI, no logo, no photorealism, no smooth gradients, seamless tile, orthographic flat texture
```

通用反向提示词：

```text
photorealistic, 3d render, blurry, soft brush, text, letters, watermark, logo, UI frame, item icon border, excessive noise, high resolution realistic photograph, plastic look, glossy fantasy gem overload, copied Minecraft texture
```

方块贴图补充：

```text
seamless square block texture, tileable on all edges, balanced contrast, distinct from vanilla Minecraft while compatible with voxel resource packs
```

物品贴图补充：

```text
single item sprite on transparent background, centered, readable silhouette, no cast shadow, no background
```

机器贴图补充：

```text
industrial voxel machine block texture, functional front face, bolts, vents, input slot details, subdued realistic material colors, readable 16x16 sprite
```

本项目已加入自动化脚本：

```powershell
python tools\generate_agnes_item_textures.py
```

脚本会从 `C:\Users\du_ji\WorkBuddy\agnes\providers.json` 读取 Agnes 配置，调用 `agnes-image-2.1-flash`，把原始生成图保存到 `tmp/imagegen/agnes-modern-chemistry/`，再裁切缩放为 `assets/earth_online/textures/item/*.png`。不要把 API Key 写进仓库或日志。

```text
Constraints: isolated icon only, generous padding, crisp edges, suitable for downscaling to 16x16.
```

## 使用建议

- 方块贴图优先生成 `32x32` 或 `64x64` 草稿，再人工缩到 `16x16` 调整像素可读性。
- 矿物同系列要先固定调色板，例如磁铁矿全系使用黑、蓝灰、深灰，不要每张图都随机偏色。
- 矿石贴图的矿物占比要对应纯度：贫矿稀疏，普通矿适中，富矿密集。
- 机器贴图要先生成正面，再用同风格扩展侧面、顶部、底部。
- 不要让 AI 生成文字、化学式或 UI 边框，化学式应由语言文件和 tooltip 显示。
